# GTranslate Clone

A clone of Google Translate using React, Vite, and Jotai.

## Features

- [x] Speech-to-Text (STT) - Google WebkitSpeechRecognition
- [x] Text-to-Speech (TTS) - Google Cloud Text-to-Speech
- [x] Translation - Google Translation API

## Development

```bash
pnpm install
pnpm dev
```

## Deployment

```bash
pnpm build
pnpm deploy
```
